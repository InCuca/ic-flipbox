# ic-flipbox

> A flipbox div with option for a button

# Usage

    <ic-flipbox text="Hello World!"></ic-flipbox>

## Installation

### Using yarn

`yarn add ic-flipbox`

### Using npm

`npm i --save ic-flipbox`

## Demo and Docs

`npm run serve`
