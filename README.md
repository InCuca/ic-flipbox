# ic-flipbox

> A flipabble element

# Usage

    <ic-flipbox>
      <div slot="front-content">My Front Content</div>
      <div slot="back-content">My Back Content</div>
    </ic-flipbox>

## Installation

### Using yarn

`yarn add ic-flipbox`

### Using npm

`npm i --save ic-flipbox`

## Demo and Docs

`npm run serve`
